- Secton 2 : TypeScript Basis & Basic Types
	- Type Assignment & Type Inference #typescript/types
		- ```ts
		  // 多餘的 code , typescript 本身就會自行推斷 "name" 的型別
		  // 不需要指定 string 給它
		  const name:string = "name"
		  // 正確
		  const name = "name"
		  // 只有宣告變數未賦予值，才需要給予型別
		  let name2:string
		  name2="name"
		  ```
