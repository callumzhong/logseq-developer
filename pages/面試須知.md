	- ### 變數差異
		- `var`：全域變數會汙染 window，常見問題 callback function
		  ```js
		  for ( var i = 0 ; i < 3 ; i++ ) {
		    	console.log(i)
		  	setTimeout( function() { console.log(i) } , 0 );
		  }
		  // 0
		  // 1
		  // 2
		  // 3
		  // 3
		  // 3
		  ```
			- 註冊 callback function 輸出變數 i 傳遞給 setTimeout 執行
			- setTimeout 是異步處理，當 For Loop 執行完才會開始執行
			- 因為全域變數的關係，這時候
