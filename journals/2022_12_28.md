- ```css
   /* In Chrome, Safari, Opera, and other WebKit-based browsers */
    body::-webkit-scrollbar {
      display: none;
    }
  
    /* In IE (Internet Explorer) and Edge */
    body {
      -ms-overflow-style: none;
    }
    /* In Firefox */
    html {
      scrollbar-width: none;
    }
  ```
  > 使用 CSS 隱藏滾動條 (不影響滾動功能)
